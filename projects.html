INDIVIDUAL PROJECTS

CAPSTONE PROJECT

This project studies the main factors that influence consumers' intention to use smart home security devices in urban areas of Malaysia. It aims to understand adoption behavior and provide insights for technology providers and policymakers to promote the use of smart home technologies for smarter living environments.

SOCIAL AND WEB ANALYTICS

This project uses Python (NetworkX) to perform network analysis on house finch social interaction data, identifying key birds and connections that significantly influence the spread of the Mycoplasma gallisepticum disease.

PROGRAMMING PRINCIPLES

This project is about developing a Python-based Classroom Management System designed to help teachers, students, and administrators manage student records, attendance, and grades efficiently. The system reads and processes student information from files, validates inputs, retrieves current date and time, and provides a user-friendly interface for classroom data management.

GROUP PROJECTS

MALAYSIA AIRLINES PROJECT

This project focuses on improving the Malaysia Airlines system, particularly the check-in process, live chat support and app login system. We developed a Figma prototype that introduces several enhancements, including a display of upcoming flights on the check-in page, integration of a 24-hour AI-powered live chatbot, and the addition of an extra verification method on the login page.

ARTIFICIAL INTELLIGENCE

This project focuses on implementing the A* search algorithm to collect all treasures using the most efficient path, minimizing both the total steps taken and total energy used, while avoiding all traps and obstacles. Players can utilize rewards to ease their journey.

ADVANCED BUSINESS ANALYTICS

This project focuses on developing predictive models and identifying the best-performing one among several machine learning algorithms, including K-Nearest Neighbours (KNN), Random Forest, Logistic Regression and XGBoost, using a financial fraud dataset. Power BI is also utilized to visualize data patterns related to financial fraud.

ENTERPRISE ARCHITECTURE

The project is about developing an enterprise architecture plan for Buyomart to support its expansion by upgrading its POS, inventory, delivery integration and sales tracking systems using modern cloud-based and AI-driven technologies.

DATA STRUCTURE AND ALGORITHMS

The project is about developing interactive visualizations of Binary Search Tree, AVL Tree and B-Tree using Python to demonstrate how each data structure stores and manages data efficiently through insertion, deletion and balancing operations.

STATISTICS FOR DATA ANALYTICS

This project uses SAS Enterprise Guide to perform statistical modeling and predictive analysis on housing data, identifying key factors such as age，location, and amenities that significantly impact house prices.
