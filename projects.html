BUSINESS INTELLIGENCE

This project focuses on predicting short-term retail sales to optimize inventory management and resource allocation. To achieve this, our team analyzed a four-year superstore dataset, handling missing values and mitigating extreme outliers through 99th-percentile winsorization using SAS Miner. We then developed, tuned, and compared Random Forest and Gradient Boosting predictive models using SAS Miner. By evaluating the models, we selected Gradient Boosting for its superior accuracy and conducted feature importance analysis, which revealed that specific product sub-categories—rather than geographic locations—are the primary drivers of sales. Ultimately, we translated these predictive findings into actionable, data-driven recommendations for targeted promotions and product-based business strategies.

DATABASE MANAGEMENT SYSTEMS

This project focuses on evaluating and optimizing database query performance and storage efficiency for large-scale retail transaction data. To achieve this, our team conducted a comparative experimental analysis of different indexing strategies—specifically B-Tree, Hash, and non-indexed tables—across both relational (MySQL) and NoSQL (MongoDB) environments. We systematically measured query latency and storage overhead across various retrieval patterns, including point lookups, range filtering, sorting, and aggregations, using SQL and MQL within MySQL Workbench and MongoDB Shell. By profiling these executions, we discovered that B-Tree indexing provided the most optimal balance, delivering consistently fast execution times while significantly minimizing storage overhead compared to Hash indexing. Ultimately, we translated these empirical findings into actionable database design recommendations that help retail businesses reduce query latency, improve operational effectiveness, and optimize cloud storage costs.

DATA MINING AND KNOWLEDGE DISCOVERY

This project forecasts California wildfire risks to support early warning systems and disaster mitigation. Using SAS Enterprise Miner, we preprocessed over 128,000 meteorological records from 262 weather stations, handling data skewness, anomalies, and imbalanced target classes. We then built and evaluated complex Ensemble architectures against High-Performance (HP) Neural Networks. The HP Neural Network proved optimal, achieving 96.77% accuracy and superior recall by capturing non-linear relationships in key factors like solar radiation, soil temperature, and relative humidity. Ultimately, these findings provided actionable recommendations, enabling authorities to optimize UAV patrols, issue targeted red-flag warnings, and strategically pre-position firefighting resources in high-risk zones.
